# sms-activate-platform

接码短信平台

### 环境要求

- nodejs版本: v14.21.3
- yarn版本: 1.22.22

### 安装依赖

```shell
yarn install
```

### 启动

```shell
yarn serve
```

### 访问

http://localhost:8080

## 页面

- 首页: http://localhost:8080/
- 登录: http://localhost:8080/login
- 充值页面: http://localhost:8080/payment/index.html

### 打包

```shell
yarn build
```
