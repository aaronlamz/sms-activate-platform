# 接码短信平台

### 环境要求

- nodejs版本: v14.21.3
- yarn版本: 1.22.22

### 安装依赖

```shell
yarn install
```

### 启动项目

```shell
yarn serve
```

### 本地访问

http://localhost:8080

## 页面结构

- 首页: http://localhost:8080/
- 登录: http://localhost:8080/login
- 充值页面: http://localhost:8080/payment/index.html

### 打包部署

```shell
yarn build
```
